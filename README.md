🛍️ Fashion Hub - Flask Cloth Store

Fashion Hub is a beginner-friendly **Clothing Store Web Application** built using **Python Flask**. The project demonstrates how to create a simple e-commerce homepage with dynamic product filtering and searching using Flask.

Users can browse clothing products, search for items by name, filter products by category, and sort them by price. The application uses Python lists to store product data and Jinja2 templates to dynamically display products on the webpage.

## ✨ Features

* 🔍 Search clothes by product name
* 👕 Filter products by category (Men, Women)
* 💰 Sort products by price (Low to High / High to Low)
* 🖼️ Display product images with details
* 📱 Responsive and user-friendly interface
* ⚡ Built using Flask and Jinja2 templating

## 🛠️ Technologies Used

* Python
* Flask
* HTML5
* CSS3
* Jinja2

## 🚀 How to Run

1. Clone the repository.
2. Install Flask:

   ```bash
   pip install flask
   ```
3. Run the application:

   ```bash
   python app.py
   ```
4. Open your browser and visit:

   ```
   http://127.0.0.1:5000/
   ```

This project was created to practice:
